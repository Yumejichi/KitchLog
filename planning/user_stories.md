# User Stories

Reference the Writing User Stories final project guide in the course portal for more information about how to complete each of the sections below.

## Outline User Roles

Home Cook – A user who wants to save recipes, organize meals, and generate grocery shopping lists.

## Draft User Stories

1. As a Home Cook, I want to create and save recipes, so that I can keep track of meals I enjoy making.

2. As a Home Cook, I want to edit a recipe, so that I can keep the ingredients and instructions accurate and up to date.

3. As a Home Cook, I want to delete a recipe, so that I can remove recipes I no longer use.

4. As a Home Cook, I want to paste a recipe link and have the app automatically create a recipe, so that I don't have to manually enter the ingredients and instructions.

5. As a Home Cook, I want to browse all my saved recipes, so that I can quickly decide what to cook.

6. As a Home Cook, I want to search and filter recipes by tags or categories, so that I can easily find recipes that match my preferences.

7. As a Home Cook, I want to mark recipes as favorites, so that I can quickly access the recipes I make most often.

8. As a Home Cook, I want to select multiple recipes for a cooking plan, so that I can organize the meals I want to prepare.

9. As a Home Cook, I want the app to automatically generate a grocery list from my selected recipes, so that I don't have to manually write one.

10. As a Home Cook, I want to view detailed recipe information, including ingredients and instructions, so that I can follow the recipe while cooking.

11. As a Home Cook, I want to mark grocery items as purchased, so that I can keep track of what I've already bought.

12. As a Home Cook, I want to uncheck an item if I still need to buy it, so that I can easily update my shopping list.
